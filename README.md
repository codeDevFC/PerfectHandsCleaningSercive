# Perfect Hands Cleaning Service Website

A professional website for Perfect Hands Cleaning Service built with React, styled-components, and Framer Motion.

## Features

- Responsive design for all screen sizes
- Animated hero section with image carousel
- Interactive service cards
- WhatsApp integration for direct customer contact
- Contact form that sends inquiries directly to WhatsApp
- Comprehensive coverage area information
- Mobile-friendly navigation

## Technology Stack

- React.js
- styled-components for styling
- Framer Motion for animations
- React Router for navigation
- React Icons for UI icons
